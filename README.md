# 味享厨 CookShare —— 智能 AI 菜谱助手

## 一、项目概述

味享厨（CookShare）是一款基于人工智能技术的新一代智能菜谱平台。我们致力于解决用户“不知道吃什么”、“有食材不会做”、“想吃得健康又美味”的日常烹饪痛点，通过自然语言理解、大模型推理与营养知识图谱，为用户提供个性化、可执行、高可信度的菜谱建议。

## 二、核心功能

### 1. AI 智能菜谱生成
用户只需输入手边有的食材、饮食偏好及特殊需求，系统即可在秒级内生成包含菜名、简介、所需食材清单、分步图文/视频操作指南及营养成分分析的完整菜谱。

### 2. 多模态内容呈现
支持文本、图片、短视频等多种形式展示菜谱步骤；集成语音播报，实现“边听边做”。

### 3. 个性化推荐引擎
基于用户历史行为、口味偏好、健康目标，持续优化推荐结果；支持家庭成员画像管理。

### 4. 社区分享与互动
用户可保存、收藏、分享自己的 AI 生成菜谱；支持上传实拍成果，形成“AI + UGC”闭环。

## 三、技术亮点

- **大模型驱动**：基于通义千问（Qwen）等国产大模型微调，确保中文烹饪语境理解准确。
- **知识增强**：融合专业营养数据库与百万级真实菜谱数据，提升生成内容的专业性与可行性。
- **云原生架构**：部署于阿里云，支持弹性伸缩，保障高并发下的稳定体验。
- **隐私安全**：用户数据本地加密，不存储敏感信息，符合GDPR/个人信息保护法要求。

## 四、目标用户

- 忙碌的上班族（快速决策晚餐）
- 健康饮食关注者（控糖、减脂、健身人群）
- 烹饪新手（需要详细指导）
- 家庭主妇/主夫（寻找新菜式灵感）

## 五、当前进展与下一步计划
