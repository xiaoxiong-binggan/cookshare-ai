🌟 项目简介
在快节奏的都市生活中，外卖成了日常，但厨房的烟火气才是生活的温度。CookShare 是一个轻量级、全前端的菜谱创作与分享 Web 应用，专为“想做饭但不知从何开始”的你设计。

✨ 支持图文+短视频菜谱创作：上传步骤图、录制烹饪小视频，一键生成专属菜谱。
🌍 全球加速 & 安全托管：基于 阿里云 ESA Pages 部署，依托边缘节点实现毫秒级加载与 DDoS 防护。
⚡ 纯前端实现，零后端依赖：利用浏览器本地存储（localStorage）暂存内容，快速原型，即开即用。
🎯 响应式设计：手机、平板、电脑全适配，厨房里也能轻松查看菜谱！
本项目深度兼容 React / Vue 等主流框架（当前为原生 HTML + TypeScript 实现），充分展现 前端工程 + 边缘计算 + 用户体验 的融合创新。

🚀 为什么选择 CookShare？
对抗“外卖依赖”：用简单、有趣的交互，降低做饭门槛，唤醒你的厨艺潜能。
边缘即体验：借助阿里云 ESA Pages 的全球边缘网络，无论你在东京、伦敦还是新加坡，菜谱加载都快如闪电。
AI 原生起点：未来将集成 AI 菜谱生成、食材识别、步骤语音指导等能力，打造真正的“厨房智能助手”。
🛠 技术栈
前端：HTML5 + TypeScript + Tailwind CSS（轻量高效）
部署平台：阿里云 ESA Pages（边缘安全加速 + 全球 CDN）
存储：浏览器 localStorage（演示阶段），后续将接入云端存储
兼容性：支持 PWA，可安装为桌面/手机应用
