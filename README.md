一、项目概述
味享厨（CookShare）是一款以 AI 为核心驱动力的新一代智能菜谱平台，聚焦解决用户「食材闲置不知如何搭配」「烹饪步骤复杂难上手」「健康饮食难兼顾美味」等日常痛点。
我们通过自然语言理解 + 专业烹饪知识图谱 + 多模态内容生成，为用户提供「从食材到成品」的全链路解决方案 —— 只需输入手边食材、口味偏好或健康需求，即可秒级获得可执行、个性化、高可信度的完整菜谱，让烹饪从 “难题” 变成 “乐趣”。
二、核心功能
1. AI 智能菜谱生成（核心亮点）
极简输入：支持自然语言描述（如 “我有鸡蛋、番茄，想做快手菜”“减脂期，有鸡胸肉怎么吃”）
完整输出：生成包含「菜名 + 创意简介 + 精准食材清单 + 分步图文指南 + 营养成分表」的全维度菜谱
动态适配：自动匹配用户厨房工具（如 “无烤箱版替代方案”）、饮食禁忌（如 “清真 / 素食适配”）
2. 多模态沉浸式体验
视听双辅：支持步骤语音播报（边听边做，解放双手）
多形式呈现：文本 / 高清示意图 / AI 生成短视频（后续新增漫画风格视频，可选日系 / 国漫等视觉风格）
3. 个性化推荐与管理
用户画像驱动：基于历史浏览、收藏、反馈，推荐符合口味偏好的新菜式
家庭化管理：支持添加 “儿童 / 老人 / 健身者” 等家庭成员标签，自动调整菜谱的口味 / 营养配比
4. 社区互动闭环
UGC + AI 融合：用户可分享 AI 生成菜谱，也能上传 “实拍成品 + 改良笔记”
社交化激励：点赞、收藏、评论功能，优质内容可获得 “星级厨师” 标识
三、技术亮点
大模型 + 知识增强：基于通义千问（Qwen）微调，融合百万级真实菜谱库与专业营养数据库，既保证 “懂中文烹饪语境”，又确保内容专业可行
云原生高可用：部署于阿里云 ESA Pages，支持弹性伸缩，保障高并发场景下的稳定访问
隐私安全合规：用户数据本地加密存储，不收集敏感信息，完全符合《个人信息保护法》与 GDPR 规范
前端体验优化：采用 React + TypeScript 开发，响应式布局适配 PC / 移动端，交互流畅无卡顿
四、目标用户
忙碌上班族：快速搞定 “今天吃什么”，30 分钟内完成晚餐
健康饮食人群：控糖、减脂、健身党，精准匹配营养需求
烹饪新手：0 基础也能跟着 “分步指南 + 语音” 做出可口饭菜
家庭主厨：寻找新菜式灵感，兼顾全家口味与健康
